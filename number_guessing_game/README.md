# Number Guessing Game

A beginner-friendly Python game where the user tries to guess a randomly generated number.

## Features
- Random number generation using Python's `random` module
- Input validation to prevent crashes
- Replay option to play multiple rounds
- Tracks attempts and high score
- Modular functions for clean, reusable code

## Skills Demonstrated
- Loops & nested loops
- Conditional statements
- Functions & modular design
- Error handling (`try/except`)
- State tracking (variables for attempts and high score)
