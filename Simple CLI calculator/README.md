# Python CLI Calculator

A simple command-line calculator built in Python that supports basic arithmetic operations with user-friendly input handling.

## Features

- Supports addition (`+`), subtraction (`-`), multiplication (`*`), and division (`/`)
- Input validation for numbers and operations
- Special commands for easy control:
  - Type `operation` at any number prompt to change the operation
  - Type `back` at the second number prompt to re-enter the first number
- Repeated calculations without restarting the program
